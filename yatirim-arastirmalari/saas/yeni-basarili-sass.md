## 🏢 Ürün Detayları

### 1. Linear — Project Management

**Alan:** Proje yönetimi / Issue tracking
**Kuruluş:** 2019
**Founder:** Karri Saarinen (eski Airbnb)
**Yatırımcılar:** a16z, Sequoia
**Toplanan Yatırım:** $35M

**Problem:** Jira çok büyük, yavaş, Agile için karmaşık. Geliştiriciler daha hızlı, daha basit bir şey istiyor.

**Çözüm:** "GitHub ile entegre, sadece keyboard shortcut'larla çalışan, modern UI" — 10x daha hızlı issue yönetimi

**Neden Başarılı?**
- Gelştiriciler arasında "en sevilen" proje yönetim aracı
- Keyboard-first tasarım — mouse'a gerek yok
- GitHub, Figma, Slack ile native entegrasyon
- Viral: "Linear'ın UI'sunu kopyalamak" bir trend oldu

**Kullanıcı Sayısı:** 5,000+ şirket (Netflix, Spotify, Stripe, GitHub)
**Revenue:** $50M+ ARR tahmini

**Fiyatlandırma:**
| Plan | Fiyat | Özellikler |
|------|-------|------------|
| Free | $0 | 250 issue |
| Pro | $12/kullanıcı/ay | Sınırsız |
| Organization | $20/kullanıcı/ay | Admin, analytics |

**Herkesin Kaçırdığı Nokta:** Linear sadece "hızlı Jira" değil — aslında bir "developer experience" ürünü. Issue'larınızı kanban, list, roadmap olarak görebiliyorsunuz. En önemlisi: her şey keyboard ile yapılıyor. Bu, geliştiricilerin "flow state" kalmasını sağlıyor.

### 2. Notion — All-in-one Workspace

**Alan:** Workspace / Dokümantasyon / Wiki / CRM
**Kuruluş:** 2013
**Founder:** Ivan Zhao (eski AngelList)
**Yatırımcılar:** a16z, Sequoia, Coatue
**Toplanan Yatırım:** $275M (Seri D)
**Kullanıcı Sayısı:** 30M+ (2024)
**Revenue:** $200M+ ARR (2024)

**Problem:** Şirketler onlarca araç kullanıyor — Confluence, Google Docs, Asana, Monday, Trello. Hepsi farklı yerde, bilgi kayıyor.

**Çözüm:** "Notion'la her şeyi bir yerde yap" — dokümantasyon, proje yönetimi, wiki, CRM, notes, database

**Neden Başarılı?**
- "Her şeyi yapabilen" esneklik
- Templates — hazır şablonlarla başlama
- AI özellikleri (Notion AI)
- Topluluk — kullanıcılar paylaşıyor

**Fiyatlandırma:**
| Plan | Fiyat | Özellikler |
|------|-------|------------|
| Free | $0 | Bireysel |
| Plus | $16/kullanıcı/ay | 30+ kullanıcı |
| Business | $25/kullanıcı/ay | Analytics, SAML |
| Enterprise | Custom | Özel |

**Herkesin Kaçırdığı Nokta:** Notion sadece "note-taking app" değil — aslında bir "no-code platform." Formüller, veritabanı ilişkileri, API entegrasyonları ile tam teşekküllü uygulamalar build edilebiliyor. KOBİ'ler Notion'ı CRM olarak kullanıyor.

### 3. Figma — Collaborative Design

**Alan:** UI/UX Design
**Kuruluş:** 2012 (Figma), 2020 (Figma DNA)
**Founder:** Dylan Field, Evan Wallace
**Yatırımcılar:** a16z, Sequoia, Greylock
**Toplanan Yatırım:** $200M+
**Kullanıcı Sayısı:** 4M+ tasarımcı
**Revenue:** $400M+ ARR
**Değerleme:** $10 Milyar+

**Problem:** Sketch, Photoshop — tek kişi tasarımı yapıyor, takım çalışması zor. Dosyaları paylaşmak, versiyon kontrolü derdi.

**Çözüm:** "Tarayıcıda çalışan, real-time işbirliği" — aynı dosyada birden fazla kişi aynı anda çalışıyor

**Neden Başarılı?**
- Tarayıcıda çalışıyor — kurulum yok
- Real-time işbirliği — "Google Docs" gibi
- Developer mode — geliştiriciler için
- Plugin ekosistem — 500+ eklenti

**Fiyatlandırma:**
| Plan | Fiyat | Özellikler |
|------|-------|------------|
| Free | $0 | 3 projeler |
| Professional | $15/kullanıcı/ay | Sınırsız projeler |
| Organization | $45/kullanıcı/ay | Kütüphane paylaşımı |
| Enterprise | Custom | Özel |

**Herkesin Kaçırdığı Nokta:** Adobe Figma'ı $20 Milyara satın almak istedi — ABD mahkemesi engelledi. Sebep: rekabeti kısıtlayacak. Bu, Figma'nın pazar değerinin ne kadar yüksek olduğunu gösteriyor. Artık sadece "tasarım aracı" değil, "design system platformu."

### 4. Vercel — Frontend Cloud

**Alan:** Web hosting / Frontend deployment
**Kuruluş:** 2015
**Founder:** Guillermo Rauch (eski LearnBoost)
**Yatırımcılar:** a16z, GV, Tiger Global
**Toplanan Yatırım:** $260M
**Kullanıcı Sayısı:** 1M+ projeler
**Revenue:** $100M+ ARR tahmini

**Problem:** Web sitesi deploy etmek karmaşık — server kur, Nginx ayarla, CDN yapılandır. Hız için ayrı optimizasyon.

**Çözüm:** "Git'e push et, site yayında" — Next.js'in kurucusunun firması. Edge network, otomatik SSL, preview deploy'lar.

**Neden Başarılı?**
- Next.js ile native entegrasyon
- "Zero config" — çalışıyor
- Preview URL'ler — her PR için canlı demo
- Edge network — global hız

**Fiyatlandırma:**
| Plan | Fiyat | Özellikler |
|------|-------|------------|
| Hobby | $0 | 100GB bant |
| Pro | $20/ay | 1TB, öncelikli |
| Enterprise | Custom | Özel |

**Herkesin Kaçırdığı Nokta:** Vercel sadece "hosting" değil — aslında bir "frontend development platform." Artık sadece deploy değil, analytics, monitoring, AI SDK'sı bile var. Web development'ın "Heroku"dan "Vercel"e geçişi tamamlandı.

### 5. Supabase — Firebase Alternatif

**Alan:** Backend-as-a-Service / Database
**Kuruluş:** 2020
**Founder:** Paul Copplestone (eski GitHub)
**Yatırımcılar:** a16z, Coatue, Y Combinator
**Toplanan Yatırım:** $80M
**Kullanıcı Sayısı:** 1M+ geliştirici

**Problem:** Firebase pahalı, Postgres alışkanlığı zor, açık kaynak değil.

**Çözüm:** "Açık kaynak Firebase" — Postgres tabanlı, açık kaynak, self-hosting mümkün, Firebase API uyumlu

**Neden Başarılı?**
- Postgres — güvenilir, SQL
- Açık kaynak — veriniz sizin
- Firebase uyumlu — migration kolay
- Realtime subscriptions

**Fiyatlandırma:**
| Plan | Fiyat | Özellikler |
|------|-------|------------|
| Free | $0 | 500MB DB |
| Pro | $25/ay | 8GB DB |
| Team | $599/ay | 100GB DB |
| Enterprise | Custom | Sınırsız |

**Herkesin Kaçırdığı Nokta:** Supabase sadece "veritabanı" değil — aslında "backend platform." Auth, storage, edge functions, realtime — Firebase'in yaptığı her şeyi yapıyor. Üstelik Postgres'in tüm yetenekleriyle. Bu, Firebase'den kaçışın resmi.
