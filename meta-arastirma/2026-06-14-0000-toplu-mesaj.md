## WhatsApp Business API — Toplu Mesaj Detayları

### Nasıl Çalışır?

1. **Template Message (Şablon Mesaj):** WhatsApp Business API'de toplu mesaj göndermek için önceden onaylanmış template mesajlara ihtiyacınız var. Bu şablonlar Meta tarafından inceleniyor ve onaylanıyor.

2. **72 Saat Kuralı:** Müşteriyle son etkileşimden itibaren 72 saat içinde serbestçe mesaj atabilirsiniz. Bu süre geçtikten sonra sadece onaylı template mesajlarla ulaşabilirsiniz.

3. **Mesaj Limiti:** WhatsApp Business API şu anda işletme hesapları için ~1000-100.000 arasında mesaj limiti sunuyor (hesap türüne göre).

### Kaç Mesaj Limiti Var, Nasıl Aşılır?

- **Standart limit:** Her işletme hesabı için günlük ~1000 mesaj
- **Yükseltme:** Meta Business Manager üzerinden başvuru yapılabilir
- **Kademeli artış:** Meta, yüksek kaliteli mesaj trafiği olan hesaplara otomatik limit artışı sağlıyor
- **Önemli:** Spam şikayetleri limiti düşürebilir — müşteri deneyimi kritik

### Örnek Kod (WhatsApp Business API)

```python
import requests
import json

# WhatsApp Business API ile mesaj gönderimi
def send_whatsapp_message(phone_number, template_name, header_vars=None, body_vars=None):
    url = f"https://graph.facebook.com/v18.0/{PHONE_NUMBER_ID}/messages"
    
    headers = {
        "Authorization": f"Bearer {ACCESS_TOKEN}",
        "Content-Type": "application/json"
    }
    
    message_data = {
        "messaging_product": "whatsapp",
        "to": phone_number,
        "type": "template",
        "template": {
            "name": template_name,
            "language": {"code": "tr"},
            "components": []
        }
    }
    
    # Header bileşeni (opsiyonel)
    if header_vars:
        message_data["template"]["components"].append({
            "type": "header",
            "parameters": [{"type": "text", "text": var} for var in header_vars]
        })
    
    # Body bileşeni (opsiyonel)
    if body_vars:
        message_data["template"]["components"].append({
            "type": "body",
            "parameters": [{"type": "text", "text": var} for var in body_vars]
        })
    
    response = requests.post(url, headers=headers, json=message_data)
    return response.json()

# Toplu mesaj gönderimi
def send_bulk_messages(phone_numbers, template_name, body_vars):
    results = []
    for phone in phone_numbers:
        result = send_whatsapp_message(phone, template_name, body_vars=body_vars)
        results.append(result)
    return results
```

### Adım 4: Python ile Toplu Mesaj Scripti Yaz
```python
# WhatsApp Business API - Toplu Mesaj Scripti
import requests
import time

ACCESS_TOKEN = "YOUR_ACCESS_TOKEN"
PHONE_NUMBER_ID = "YOUR_PHONE_NUMBER_ID"
TEMPLATE_NAME = "your_template_name"

def send_bulk_with_delay(phone_numbers, template_name, delay=1):
    """Rate limiting ile toplu mesaj gönder"""
    url = f"https://graph.facebook.com/v18.0/{PHONE_NUMBER_ID}/messages"
    headers = {
        "Authorization": f"Bearer {ACCESS_TOKEN}",
        "Content-Type": "application/json"
    }
    
    for phone in phone_numbers:
        payload = {
            "messaging_product": "whatsapp",
            "to": phone,
            "type": "template",
            "template": {
                "name": template_name,
                "language": {"code": "tr"}
            }
        }
        
        try:
            response = requests.post(url, headers=headers, json=payload)
            print(f"Gönderildi: {phone} - {response.status_code}")
        except Exception as e:
            print(f"Hata: {phone} - {e}")
        
        time.sleep(delay)  # Rate limit aşılmaması için bekle
```
