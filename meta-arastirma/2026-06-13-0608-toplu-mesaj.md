# WhatsApp Toplu Mesaj Araştırması — 2026-06-13-0608

## Özet

WhatsApp toplu mesaj araçları ekosisteminde iki ana kategori var: resmi WhatsApp Business API çözümleri ve gayri resmi otomasyon araçları. Resmi API, kurumsal ölçekli uygulamalar için Facebook Business doğrulaması gerektiriyor ancak daha yüksek teslimat oranları ve gelişmiş analitik sunuyor. Gayri resmi araçlar düşük giriş门槛ı sağlıyor ancak hesap ban riski taşıyor.

**Herkesin Kaçırdığı Nokta:** Bugün ortaya çıkan Meta AI chatbot güvenlik ihlali (20.000+ Instagram hesabı çalındı) WhatsApp toplu mesaj bot'ları için kritik bir uyarı niteliğinde. Prompt injection saldırıları WhatsApp chatbot'larını da etkileyebilir. Kullanıcı mesajlarındaki özel karakterler bot'un davranışını değiştirebilir. Bu, toplu mesaj platformlarının da benzer güvenlik açıkları barındırabileceğini gösteriyor.

## Adım Adım Yapım Rehberi

### WhatsApp Business API ile Toplu Mesaj Gönderme (n8n + Node.js)

**1. Adım: WhatsApp Business API Erişimi**
```
- Meta Business Suite'te uygulama oluştur
- WhatsApp Business API'ye erişim iste
- Telefon numarası onayla
- Webhook URL'ini yapılandır
```

**2. Adım: n8n Workflow Kurulumu**
```javascript
// n8n'de WhatsApp node yapılandırması:
// 1. WhatsApp Business API node ekle
// 2. Credentials ekle (App ID, App Secret)
// 3. Webhook URL'i Meta'ya bildir
```

**3. Adım: Template Message Onayı**
- Her toplu mesaj için template oluştur
- Meta Business Suite'te onaylat (24-48 saat)
- Template statüsünü kontrol et

**4. Adım: CSV'den Toplu Gönderim**
```javascript
// whatsapp-bulk-sender kullanımı:
node index.js --csv contacts.csv --template "order_confirm"
```
