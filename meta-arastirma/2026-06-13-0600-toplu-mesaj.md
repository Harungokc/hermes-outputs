# WhatsApp Toplu Mesaj Araştırması — 2026-06-13 06:00

## Özet

WhatsApp toplu mesaj araçları生态系统中，存在两类截然不同的解决方案：通过官方WhatsApp Business API的合规方案，和通过WhatsApp Web界面实现的非官方方案。官方API方案更适合企业级应用，需要Facebook Business验证，但提供更高的送达率和更好的 analytics。非官方方案门槛低但存在封号风险。

**Herkesin Kaçırdığı Nokta:** WhatsApp toplu mesajda en büyük teknik engel "template message" zorunluluğu. Birdn fazla kişiye mesaj atarken, her mesaj "template" olarak onaylatılmalı. Bu süreç 24-48 saat sürebilir ve maliyetli. Ama çözüm var: önceden onaylatılmış template'leri tekrar kullanmak veya "session message" olarak müşteriyle önceden konuşmuş olanlara anında mesaj atmak.

## Adım Adım Yapım Rehberi

### Seçenek 1: WhatsApp Business API + n8n (Önerilen)

**1. Facebook Developer Account oluştur**
```
1. https://developers.facebook.com adresine git
2. "My Apps" → "Create App" → "Business" seç
3. WhatsApp Business API seç
4. Business verification tamamla (Instagram/LinkedIn doğrulaması)
```

**2. WhatsApp Business API Kurulumu**
```
1. Meta Business Suite'e git
2. WhatsApp → API Setup → Phone number ekle
3. Access Token al (Store token güvenli yerde)
4. Webhook URL set et (n8n endpoint)
```

**3. n8n Workflow Import**
```bash
# n8n'i kur
npm install n8n -g

# Workflow JSON'u import et
# n8n'de: Settings → Import from JSON
# Dosya: n8n-workflows/whatsapp-order-automation.json
```

**4. Ortam Değişkenleri**
```bash
WHATSAPP_ACCESS_TOKEN=your_token_here
WHATSAPP_PHONE_NUMBER_ID=your_phone_id
WHATSAPP_WEBHOOK_VERIFY_TOKEN=secure_random_token
```

### Seçenek 2: WhatsApp Web Otomasyonu (Düşük Maliyetli)

**Dikkat: Bu yöntem ban riski taşır — Küçük ölçekli testler için uygun**

```python
# Selenium ile WhatsApp Web otomasyonu örneği
from selenium import webdriver
from selenium.webdriver.common.keys import Keys
import time

# Chrome driver başlat
driver = webdriver.Chrome()
driver.get("https://web.whatsapp.com")

# QR kod tara, giriş yap
input("WhatsApp'a giriş yaptıktan sonra Enter'a bas...")

# Contact arama ve mesaj gönderme
def send_bulk_message(contacts, message):
    for contact in contacts:
        search_box = driver.find_element_by_xpath('//*[@id="side"]//div[@contenteditable="true"]')
        search_box.send_keys(contact)
        time.sleep(1)
        search_box.send_keys(Keys.ENTER)
        time.sleep(1)
        
        # Mesaj yaz
        msg_box = driver.find_element_xpath('//*[@id="main"]//div[@contenteditable="true"]')
        msg_box.send_keys(message)
        msg_box.send_keys(Keys.ENTER)
        time.sleep(1)
```

## Herkesin Kaçırdığı Nokta #1: Template Message Zorunluluğu

WhatsApp Business API'de toplu mesaj gönderirken, ilk mesajınız mutlaka **template message** olmalı. Template'ler:
- Facebook Business Console'dan onaylatılmalı
- Onay 24-48 saat sürebilir
- Her template için ~$0.05-0.10 ücret

**Kaçırılan Çözüm:** Eğer müşteri size önceden "session message" ile ulaşmışsa (ör: "ürün bilgisi almak istiyorum"), bu kişiye 24 saat içinde **ücretsiz** mesaj atabilirsiniz. Bu "session window" süresini kullanmak，送達率ınızı artırır ve maliyeti düşürür.

## Herkesin Kaçırdığı Nokta #2: n8n ile Sıfır Kod Otomasyon
