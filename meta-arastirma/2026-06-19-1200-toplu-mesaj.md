## Ne Yapıyor? — Nasıl Çalışıyor?

### WhatsApp Bulk Messaging Nedir?
Toplu mesaj, işletmelerin aynı anda çok sayıda müşteriye WhatsApp üzerinden kişiselleştirilmiş veya standart mesaj göndermesini sağlayan sistemdir.

**İki temel yaklaşım:**

1. **WhatsApp Business API Resmi Yol** — Meta onaylı, yüksek deliverability, template message zorunluluğu
2. **Otomasyon Araçları (Unofficial)** — WhatsApp Web veya Android istemcisi üzerinden toplu mesaj, hızlı kurulum ama WhatsApp TOS riski

### Teknik Altyapı — Resmi API Yol
```
n8n Workflow:
  1. Müşteri listesi (Google Sheets / CSV)
  2. Claude AI → Mesaj kişiselleştirme (isim, sipariş no, ürün)
  3. WhatsApp Business API → Gönderim
  4. Teslimat raporu → Dashboard
```

**Template Message Zorunluluğu:** WhatsApp Business API'de toplu mesaj göndermek için Meta onaylı "template message" kullanılır. Gönderim şablonları:
- `order_confirmation` — Sipariş onayı
- `shipping_update` — Kargo bilgisi
- `abandoned_cart_reminder` — Sepet hatırlatma
- `appointment_reminder` — Randevu hatırlatma

**72 Saat Kuralı:** Müşteri son 72 saat içinde işletmeye mesaj gönderdiyse, işletme herhangi bir şablon olmadan serbestçe mesaj gönderebilir. Bu, abandoned cart recovery için kritik avantaj sağlar.

### Unofficial Bulk Messaging Araçları
**WhatsApp-Bulk-Sender (284⭐):**
- Android veya WhatsApp Web üzerinden çalışır
- CSV ile toplu kişi listesi
- Görsel + metin mesaj desteği
- Kurulumu kolay, API'ye gerek yok

**Risk:** WhatsApp, unofficial otomasyon araçlarını tespit ettiğinde hesabı askıya alabilir. İşletmeler için resmi API daha güvenli.

## Herkesin Kaçırdığı Nokta #1 — 72 Saat Kuralı = Bedava VIP Erişim

Herkes WhatsApp template message onayı için haftalar bekliyor. Ama gerçek şu: Müşteri sepete ürün ekleyip çıktığında, 72 saat içinde WhatsApp'tan "Merhaba [isim], sepetinizi mi unuttunuz?" diye mesaj atarsanız, **hiçbir template onayına gerek yok**. Bu 72 saatlik pencere, her abandoned cart senaryosu için doğrudan satış kanalı.

**Nasıl yakalanır:** n8n + Google Sheets + Claude ile her 1 saatte bir terk edilmiş sepetleri tarayıp, 72 saat kuralını kullanarak kişiselleştirilmiş hatırlatma gönder.

## Herkesin Kaçırdığı Nokta #2 — "Toplu Mesaj" Deyince İnsanlar Sansür Olduğunu Düşünüyor

WhatsApp "spam" kelimesi yasak. Ama gerçek toplu mesaj **kişiselleştirilmiş, değer veren,.Transactional mesajdır**. Amazon sipariş bildirimi de aslında "toplu mesajdır" — tek tek gönderilir ama içerik kişiselleştirilmiştir.

**Doğru toplu mesaj formülü:**
```
[Kişisel selamlama] + [Spesifik bilgi] + [Net CTA] + [Opt-out seçeneği]
```
**Yanlış:** "KAMPANYA! %50 İNDİRİM! ŞİMDİ AL!" — Bu spam, WhatsApp engelleyebilir.
