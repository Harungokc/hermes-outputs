## WhatsApp Business API — Toplu Mesaj Detayları

### Resmi API ile Toplu Mesaj

WhatsApp Business API üzerinden toplu mesaj göndermek için **Template Message** zorunluluğu devam ediyor:

1. **Template oluştur** — WhatsApp Business Manager'dan onaylat
2. **Alıcı listesi hazırla** — Telefon numaraları
3. **API çağrısı yap** — `POST /v1/messages` endpoint'i

**Limitler:**
- 24 saat içinde bir alıcıya sadece 1 template mesaj
- Rate limit: dakikada ~80 mesaj (standard), ~200 (Enterprise)
- Ses: Session başına 15-25 mesaj

### Toplu Mesaj Kazanımları (E-ticaret İçin)

- **Sepet terk hatırlatması:** 1s / 24s / 72s otomatik WhatsApp hatırlatma → **%10-15 ek dönüşüm**
- **Normal sepet terk oranı:** %70
- **Hatırlatma sonrası:** %40'a düşüyor
- **FIFA Dünya Kupası 2026 fırsatı:** Maç saatlerinde otomatik hatırlatma + ürün önerisi

### Kaçınılması Gereken: Kişiselleştirme Olmadan Toplu Mesaj

- Toplu mesaj = kişiselleştirilmiş mesaj değil
- "Merhaba [Ad]" → dinamik alan kullanılmalı
- Aksi halde WhatsApp spam olarak işaretlenir ve hesap ban riski

**Türkiye etkisi:** Şu an yok, ancak AB GDPR + WhatsApp Business API kısıtlamaları zaten mevcut. Türkiye'de KVKK uyumlu mesajlaşma için hukuki danışmanlık gerekli.

## Adım Adım: WhatsApp Business API ile Toplu Mesaj

```
1. WhatsApp Business Platform'a başvur
   → developers.facebook.com/docs/whatsapp/business-platform

2. Template mesaj oluştur
   → Business Manager > WhatsApp > Templates > Create

3. WABA (WhatsApp Business Account) onaylat
   → 24-48 saat

4. API entegrasyonu
   → n8n + HTTP Request node veya
   → Python + requests library

5. Mesaj gönder
   → POST https://graph.facebook.com/v18.0/{Phone Number ID}/messages
   → Header: Authorization: Bearer {ACCESS_TOKEN}
```
