## 2. WhatsApp Business API Toplu Mesaj

### Nasıl Çalışır?
WhatsApp Business API, toplu mesaj gönderimi için **Template Message** sistemi kullanır. Her mesaj önceden onaylanmış template'ler üzerinden gider.

**Temel Limitler:**
- **Ücretsiz tier:** 1,000 mesaj/24 saat
- **Enterprise tier:** Rate limit yok (yapılandırılabilir)
- **72 saat kuralı:** Müşteriyle son 72 saatte etkileşim olmazsa sadece template mesaj gönderilebilir

### Template Message Zorunluluğu
- Gönderim: Sadece önceden onaylanmış template mesajlarla
- Karşılaşma: Müşteri yanıt verirse → serbest mesajlaşma (24 saat)
- 24 saatlik pencere kapandığında → tekrar template gerekli

### Toplu Mesaj Kod Örneği
```python
import requests

def send_bulk_whatsapp(messages, access_token, phone_number_id):
    """WhatsApp Business API ile toplu mesaj"""
    url = f"https://graph.facebook.com/v18.0/{phone_number_id}/messages"
    headers = {
        "Authorization": f"Bearer {access_token}",
        "Content-Type": "application/json"
    }
    
    for recipient in messages:
        payload = {
            "messaging_product": "whatsapp",
            "to": recipient["phone"],
            "type": "template",
            "template": {
                "name": recipient["template_name"],
                "language": {"code": "tr"},
                "components": [
                    {
                        "type": "body",
                        "parameters": [
                            {"type": "text", "text": recipient["name"]}
                        ]
                    }
                ]
            }
        }
        response = requests.post(url, json=payload, headers=headers)
        print(response.json())
```

## 3. Bulk Message Araçları

### WASender
**URL:** wasender.com
**Ücret:** Ücretsiz + Premium
**Kullanım:** Toplu WhatsApp mesajı gönderimi için Windows uygulaması

**Özellikler:**
- CSV ile toplu import
- Kişiselleştirilmiş mesaj (ad, soyad değişkenleri)
- Görsel/medya gönderimi
- Otomatik mesaj planlama

### Kişiselleştirilmiş Mesaj (Personalization)
**En önemli nokta:** "Toplu mesaj" = "kişiselleştirilmiş mesaj" demek. Her alıcıya özel:
- `{{ad}}` → müşterinin adı
- `{{ürün}}` → ilgilendiği ürün
- `{{sipariş_no}}` → sipariş numarası

### Kaç Mesaj Limitini Aşma
1. **Birden fazla phone number kullanımı** → farklı WhatsApp Business hesapları
2. **Enterprise approval** → Meta'dan özel rate limit artırımı başvurusu
3. **Kademeli gönderim** → dakikada 100-200 mesaj yerine 50-60

## 4. Herkesin Kaçırdığı Nokta

### #1: "72 Saat Kuralı" Aslında Fırsat
Herkes bu kuralı "engel" olarak görüyor. Ama aslında:
- Müşteri sepeti terk etti → 1 saat sonra WhatsApp hatırlatması = %10-15 ek dönüşüm
- 1 gün sonra → 72 saat kuralı devreye girer → template mesajla hatırlat
- 3 gün sonra → tekrar 72 saat açılır → başka bir template ile hatırlat

**Strateji:** 1s → 24s → 72s → 7 gün → 14 gün sequence'ı kurulabilir.

### #2: Bulk Message = Otomatik DM Pazarlaması
WhatsApp Business API + n8n = tam otomasyon:
- Instagram'da yeni follower → otomatik DM hoşgeldin mesajı
- Belirli hashtag'e yorum → DM ile ürün önerisi
- Story mention → DM ile sipariş linki

### #3: Kişiselleştirme = Mesajın Değerini 3x Artırıyor
Aynı ürünü satan iki işletme:
- Biri: "Sepetinizde ürün var" → %5 dönüşüm
- Diğeri: "Merhaba [Ad], [Ürün]Sepetinizde 3 gündür bekliyor. Bugün kargo edersek yarın elinizde olur." → %15 dönüşüm
