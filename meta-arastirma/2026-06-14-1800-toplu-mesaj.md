## 2. WhatsApp Business API ile Toplu Mesaj

### Nasıl Çalışır?
WhatsApp Business API, işletmelerin toplu mesaj göndermesini sağlar ancak **şablon mesaj (template message)** zorunluluğu vardır. Bu, serbest metin mesajlaşmasını sınırlar.

### Mesaj Limitleri (2026)
- **İşletme hesabı başına:** Günlük limit, hesap türüne göre değişir
- **Yeni hesaplar:** ~250-500 mesaj/gün başlangıç
- **Onaylı işletmeler:** 10,000+ mesaj/gün
- **Rate limit:** Saniyede ~20 mesaj (bazı durumlarda 40)

### Şablon Mesaj Zorunluluğu
- Gönderen tarafından başlatılan konuşmalar için **önceden onaylanmış şablon** gerekir
- Şablon onayı WhatsApp tarafından 24-48 saat içinde yapılır
- Kategoriler: `marketing`, `utility`, `authentication`

### Bulk Messaging Araçları

**WASender**
- WhatsApp toplu mesaj gönderme aracı
- Ücretsiz versiyon mevcut
- Kişiselleştirilmiş mesaj gönderebilir (ad, sipariş no. gibi)
- ⚠️ WhatsApp Business API değil, kişisel WhatsApp hesabı ile çalışır (riskli)

**WAppool, WABlaster, Chatrace** (Ücretli platformlar)
- SaaS tabanlı WhatsApp marketing
- Aylık $29-199 arası planlar
- Otomatik yanıt, chatbot özellikleri

## 3. Herkesin Kaçırdığı Nokta

### #1: Bulk Message = Kişiselleştirilmiş Mesaj Değil
Herkes "toplu mesaj" deyince spam gibi algılıyor. Ama gerçek şu: **En etkili WhatsApp bulk message, kişiselleştirilmiş olandır.** "Merhaba [Ad], siparişiniz [Sipariş No] kargoya verildi" gibi. Bu, e-posta marketing'deki mail merge'ın WhatsApp versiyonu.

### #2: 72 Saat Kuralı — Altın Pencerе
WhatsApp'ın en az bilinen kuralı: Bir müşteriyle son 72 saat içinde mesajlaştıysanız, serbest metin mesajı gönderebilirsiniz. **Bu pencereyi kullanarak otomatik takip mesajları atılabilir.** Bunu yakalayamayan şirketler, müşteriyi kaybediyor.

### #3: Sesli Mesaj Otomasyonu
Müşteri temsilcisi sesli mesaj bırakıyor → WhatsApp'a otomatik yazıya dönüştürülüyor → AI yanıt veriyor. Hiç kimse bunu konuşmuyor ama %40 müşteri tercih ediyor.

## 4. Adım Adım Toplu Mesaj Sistemi Kurulumu

### Seçenek A: WhatsApp Business API + MCP Server
```
1. Meta for Developers → WhatsApp Business API hesabı oluştur
2. Telefon numarası onayla (sabit hat veya mobil)
3. Şablon mesajlar oluştur ve onaylat
4. MCP server kur (meta-ads-mcp veya markifact-mcp)
5. n8n veya Claude Code ile otomasyonu kur
```

### Seçenek B: Üçüncü Taraf Platform (Kolay)
```
1. WASender, WAppool veya Chatrace hesabı aç
2. WhatsApp numaranızı bağla
3. Kişiselleştirme alanları tanımla ({{ad}}, {{siparis_no}})
4. CSV ile alıcı listesi yükle
5. Gönderim zamanlaması yap
```
