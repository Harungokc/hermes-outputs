# WhatsApp Business API — Toplu Mesaj Araştırması
**Tarih:** 2026-06-12 18:00
**Kaynak:** Bing News, Meta Developer Hub, WhatsApp Business Platform

---

## Özet

WhatsApp Business API, işletmelerin toplu mesaj göndermesini sağlayan resmi Meta platformudur. June 2026'da Meta Business Agent global olarak duyuruldu — bu da WhatsApp'ta AI destekli otomatik mesajlaşmayı standart hale getiriyor. Ancak toplu mesajda kritik nokta: WhatsApp'ın kendi "broadcast" özelliği ile Business API broadcast'i farklı şeyler. İlginç olan: çoğu kişi API'yi biliyor ama "template message" kuralını kaçırıyor — spam şikayeti almayan tek yol bu.

## Toplu Mesaj Nasıl Atılır — Teknik Detay

### Resmi Yöntem: WhatsApp Business API

```
POST /v1/messages
{
  "recipient_type": "individual",
  "to": ["91XXXXXXXXXX", "91XXXXXXXXXX"],
  "type": "template",
  "template": {
    "name": "order_confirmation",
    "language": {"code": "tr"},
    "components": [
      {"type": "header", "text": "Siparişiniz Onaylandı!"},
      {"type": "body", "parameters": [{"type": "text", "text": "#12345"}]}
    ]
  }
}
```

### Kritik Kural: Template Message Zorunluluğu
WhatsApp, toplu mesajları sadece "approved templates" üzerinden kabul ediyor. Bu şablonlar:
- `order_confirmation` — Sipariş onay
- `shipping_update` — Kargo bilgisi
- `appointment_reminder` — Randevu hatırlatma
- `payment_confirmation` — Ödeme onay
- `CUSTOM` — Özel şablon (onay gerektirir, 24-48 saat)

**Herkesin Kaçırdığı Nokta:** Normal mesajla toplu mesaj atmaya çalışırsanız hesabınız banlanır. Tek yol: pre-approved template kullanmak. Türkçe şablon onayı 24-48 saat sürüyor.

### Mesaj Limiti
- **Business API:** Saatlik ~100 mesaj/numara (rate limit)
- **Scale plan:** Daha yüksek limit mümkün
- **Aşma:** IP ban — yeni numara alma gerekiyor

## Adım Adım Yapım Rehberi

### 1. WhatsApp Business API Kurulumu
```
1. https://business.facebook.com adresine git
2. "Business Settings" > "WhatsApp Accounts" > "Add"
3. Telefon numarası ekle (yeni numara önerilir)
4. API credentials al (Phone Number ID, Access Token)
5. Webhook endpoint ayarla
```

### 2. n8n ile Entegrasyon
```
1. n8n'de "HTTP Request" node ekle
2. Method: POST
3. URL: https://graph.facebook.com/v18.0/Phone_Number_ID/messages
4. Headers:
   - Authorization: Bearer ACCESS_TOKEN
   - Content-Type: application/json
5. Body (template mesaj):
   {
     "messaging_product": "whatsapp",
     "to": ["905XXXXXXXXX"],
     "type": "template",
     "template": {
       "name": "hello_world",
       "language": {"code": "tr"}
     }
   }
```

### 3. Otomatik Yanıt (AI ile)
```
1. WhatsApp Business Platform > "AI Agent" bölümüne git
2. "Create Agent" > Scenarios seç (müşteri hizmetleri, satış, randevu)
3. Knowledge base ekle (FAQ, ürün bilgileri)
4. Test et, canlıya al
```
