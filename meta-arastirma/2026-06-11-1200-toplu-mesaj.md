# WhatsApp Business API - Toplu Mesaj Araştırması
**Tarih:** 2026-06-11 12:00
**Kaynak:** Meta Business Blog, WhatsApp Business API Docs, GitHub

---

## Özet

WhatsApp Business API ile toplu mesaj gönderimi, Meta'nın resmi çözümü olan WhatsApp Business Platform üzerinden yapılıyor. 2024'teki güncellemelerle template mesaj yapısı zorunlu hale geldi ve spam koruması artırıldı. Türkiye'de direct marketing için kullanım sınırlı.

---

## Bulunan Araçlar ve Linkler

### 1. WhatsApp Business Platform (Resmi)
- **Ne iş yapıyor:** Resmi API ile toplu mesaj gönderimi, template onayı, müşteri yönetimi
- **Link:** https://business.whatsapp.com/
- **Ücret:** Konuşma başına ücretlendirme (template bazlı)
- **Limit:** Her hesap için günlük mesaj limiti mevcut (spam koruması)

### 2. Twilio WhatsApp API
- **Ne iş yapıyor:** Twilio altyapısıyla WhatsApp mesaj gönderimi
- **Link:** https://www.twilio.com/whatsapp
- **Ücret:** ~$0.05-0.10/mesaj ( hacme göre değişir)
- **Kullanıcı sayısı:** Binlerce işletme

### 3. MessageBird (Now Called MessageBird)
- **Ne iş yapıyor:** WhatsApp Business API entegrasyonu, çoklu kanal desteği
- **Link:** https://messagebird.com/whatsapp
- **Ücret:** Hacim bazlı, $0.12-0.20/mesaj
- **Kullanıcı sayısı:** 10,000+ işletme

### 4. Infobip
- **Ne iş yapıyor:** WhatsApp, SMS, RCS çoklu kanal iletişim platformu
- **Link:** https://www.infobip.com/whatsapp
- **Ücret:** $0.05-0.15/mesaj
- **Kullanıcı sayısı:** 5000+ işletme

### 5. Unofficial WhatsApp API Libraries (GitHub)
- **WhatsApp Web API (Baileys):** https://github.com/WhiskeySockets/Baileys
  - Stars: 10K+
  - Ücretsiz, unofficial
  - Kişisel kullanım için uygun
- **wwebjs:** https://github.com/wwebjs/wwebjs
  - Stars: 3K+
  - Node.js wrapper
  - Ban riski yüksek

### Risk Uyarısı
⚠️ WhatsApp, unofficial API kullanımını tespit ederse hesabı kalıcı olarak banlıyor. İş kullanımı için resmi API öneriliyor.

## Adım Adım Yapım Rehberi

### Resmi WhatsApp Business API Kullanımı

```
1. WhatsApp Business Account oluştur (Meta Business Suite)
2. Phone Number ID al
3. Template mesaj oluştur ve onaylat
4. Graph API üzerinden mesaj gönder
```

### Graph API Mesaj Gönderim Kodu Örneği
```javascript
curl -X POST https://graph.facebook.com/v21.0/PHONE_NUMBER_ID/messages \
 -H "Authorization: Bearer ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "messaging_product": "whatsapp",
    "to": "905XXXXXXXXX",
    "template": {
      "name": "hello_world",
      "language": { "code": "tr" }
    }
  }'
```

### n8n ile WhatsApp Otomasyonu
```
1. n8n-MCP server kurulumu
2. WhatsApp Business API node'u ekle
3. Trigger: Instagram DM → WhatsApp mesajı
4. Template yönetimi: Claude ile dinamik template
```

## WhatsApp Toplu Mesaj Limitleri (2026)

| Hesap Türü | Günlük Limit | Açıklama |
|------------|-------------|----------|
| Normal Business | 1000-5000 | Spam riski düşük |
| Yüksek Hacim |10,000+ | Meta onayı gerekli |
| Enterprise | Sınırsız | Özel anlaşma |

### Limit Aşımı Durumunda
- Hesap geçici olarak askıya alınır
- 24-48 saat sonra otomatik açılır
- Sürekli aşım hesabın kalıcı banına yol açabilir
