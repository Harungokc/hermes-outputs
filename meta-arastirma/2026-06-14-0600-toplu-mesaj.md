## 1. Özet

WhatsApp Business API ile toplu mesaj gönderimi için birçok açık kaynak ve ticari araç mevcut. WhatsApp'ın 24 saatlik müşteri etkileşim penceresi kuralı, toplu pazarlama mesajları için template message zorunluluğunu getiriyor. Bu kural, "broadcast" tipi mesajların sadece önceden onaylanmış template'lerle gönderilebilmesini gerektiriyor.

## 3. Herkesin Kaçırdığı Nokta #1 — 24 Saat Kuralı ve Template Message Zorunluluğu

**Herkes ne konuşuyor?** "WhatsApp'ta toplu mesaj atan araçlar" ve bunların ne kadar kolay olduğu.

**Ama gerçekte ne var?** WhatsApp Business API'de, müşteri sizinle son 24 saat içinde etkileşime girmediyse, ona ancak önceden onaylanmış bir **template message** ile ulaşabilirsiniz. Bu template'ler Meta tarafından inceleniyor ve onaylanması 24-48 saat sürebiliyor.

**Pratik sonuç:** "Kişiselleştirilmiş toplu mesaj" = her alıcıya ayrı template + her template için ayrı onay süreci. Yani "bedava toplu mesaj" diyen araçlar aslında sadece 24 saat içinde size mesaj atan müşterilere çalışıyor.

**Kaç mesaj limiti var?**
- WhatsApp Business API: Saniyede ~20 mesaj/saniye limit (business hesabı tier'ına göre değişir)
- Kurumsal hesaplarda bu limit artırılabilir

## 4. Herkesin Kaçırdığı Nokta #2 — WhatsApp'a Doğrudan Erişim ile API Arasındaki Fark

**İki farklı yöntem var:**

1. **Unofficial API (browser-based):** whatsapp-bulk-messenger, wbm gibi araçlar. WhatsApp Web'inizi kullanarak çalışıyor. UC Berkeley araştırmasına göre bu yöntemler WhatsApp'ın hizmet şartlarını ihlal ediyor ve hesap banı riski taşıyor.

2. **Official WhatsApp Business API:** Meta'nın onayladığı yöntem. Cloud API veya On-Premise API seçenekleri mevcut. Yasal, güvenli, scale edilebilir.

**Herkesin kaçırdığı:** n8n-whatsapp-automation (105⭐) gibi açık kaynak projeler, aslında official WhatsApp Business API'yi kullanıyor ve yasal yollarla sipariş otomasyonu yapıyor. Bu demek ki bedava değil ama risksiz.

## 5. Herkesin Kaçırdığı Nokta #3 — "Kişiselleştirilmiş Mesaj" Artık Mümkün

Eski toplu mesaj anlayışı: "Merhaba {{isim}}, ürünümüzü inceler misin?" — herkese aynı mesaj, sadece isim değişiyor.

**Yeni yaklaşım (2026):** Template message içinde dinamik değişkenler + medya ekleri + butonlar artık destekleniyor. Örneğin:
- `{{1}}` — müşteri adı
- `{{2}}` — sipariş numarası
- `{{3}}` — ürün görseli

Bu sayede "kişiselleştirilmiş" his veren ama yasal çerçevede çalışan kampanya mesajları oluşturulabiliyor.

## 6. Adım Adım Yapım Rehberi — WhatsApp Business API + n8n ile Toplu Mesaj

### Gereksinimler
1. WhatsApp Business hesabı
2. Meta Business Manager hesabı
3. WhatsApp Business API erişimi (Cloud API önerilen)
4. n8n kurulumu (self-hosted veya cloud)

### Adımlar

**1. WhatsApp Business API Kurulumu:**
```
1. Meta Business Manager'da uygulama oluştur
2. WhatsApp Business API'yi seç
3. Phone Number ekle (onaylı numara)
4. API token al
```

**2. n8n Workflow:**
```
- Trigger: Google Sheets (telefon numaraları listesi)
- Node: WhatsApp API
  - Template name: {{template_name}}
  - Variables: {{1}}=isim, {{2}}=urun
- Her satır için döngü
```

**3. Template Message Onayı:**
```
1. Meta Business Manager → WhatsApp → Templates
2. Yeni template oluştur
3. Kategori: MARKETING / UTILITY / AUTHENTICATION
4. Değişkenleri tanımla: {{1}}, {{2}}...
5. Onay bekle (24-48 saat)
```
