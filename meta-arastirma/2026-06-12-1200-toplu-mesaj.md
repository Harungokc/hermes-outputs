# WhatsApp Business API — Toplu Mesaj Araştırması
**Tarih:** 2026-06-12 12:00
**Kaynak:** Bing News, Meta Developer Docs, Reuters

---

## Özet

WhatsApp Business API ile toplu mesaj gönderimi, Meta'nın yeni Business Agent AI ile birlikte artık sadece template mesajlarla sınırlı değil — ancak hâlâ katı kurallar var. WhatsApp, spam'i önlemek için mesaj başına ücretlendirme sistemine geçti (Temmuz 2025 itibarıyla). Toplu mesaj için en güvenilir yol WhatsApp Business API Cloud API, ancak doğrudan "broadcast" gibi bir özellik yok — her mesaj bireysel olarak gönderiliyor.

**Herkesin kaçırdığı nokta:** WhatsApp'ın 72 saatlik "free entry point window" kuralı — müşteri sizinle ilk mesajlaştırdıktan sonra 72 saat boyunca HİÇBİR ücret ödemeden mesaj atabilirsiniz. Bu, abandoned cart recovery için altın değerinde.

## Adım Adım Yapım Rehberi: WhatsApp Business API ile Toplu Mesaj

### Gereksinimler
1. Meta Business Manager hesabı
2. WhatsApp Business API erişimi (onay süreci ~2-3 hafta)
3. Phone number (iş telefonu, kişisel numara kullanılamaz)
4. Access Token

### Adımlar

**1. WhatsApp Business Platform'a kayıt**
```
https://business.whatsapp.com/
→ "Get Started" → Business Manager hesabı oluştur
→ Telefon numaranızı doğrulayın
→ API erişimi için başvurun (Business Platform)
```

**2. Cloud API ile Python ile toplu mesaj örneği**
```python
import requests

url = "https://graph.facebook.com/v21.0/<PHONE_NUMBER_ID>/messages"
headers = {
    "Authorization": f"Bearer <ACCESS_TOKEN>",
    "Content-Type": "application/json",
}

# Toplu mesaj gönderimi (her numara için ayrı API çağrısı gerekiyor)
phone_numbers = ["+905551234567", "+905551234568", "+905551234569"]

for phone in phone_numbers:
    data = {
        "messaging_product": "whatsapp",
        "to": phone,
        "type": "template",
        "template": {
            "name": "hello_world",  # Onaylanmış template adı
            "language": {"code": "tr"},
            "components": [
                {
                    "type": "body",
                    "parameters": [
                        {"type": "text", "text": "Müşteri Adı"}
                    ]
                }
            ]
        }
    }
    response = requests.post(url, headers=headers, json=data)
    print(f"Gönderildi: {phone} - {response.status_code}")
```

**3. Template oluşturma (WhatsApp Business Manager)**
```
Business Manager → WhatsApp → Templates → "Create Template"
→ Template adı: "hello_world"
→ Kategori: Marketing / Utility / Authentication
→ Diller: Türkçe (tr), İngilizce (en)
→ Header (opsiyonel): görsel, video, belge
→ Body: "Merhaba {{1}}, siparişiniz hazır!"
→ Buttons (opsiyonel): Quick Reply, Call to Action
→ Submit → Meta onayı (~24-48 saat)
```

**4. n8n ile Otomasyon (Daha kolay)**
```
n8n workflow:
Trigger (Schedule) → WhatsApp API node → Loop over contacts → Send template message
```

**5. Mesaj limitleri ve aşımı**
- WhatsApp hard limit koymuyor ama METRİCS takip ediyor
- Spam şikayeti = ban riski
- Mesaj kalitesi puanı düşükse API erişimi kısıtlanabilir
- Önerilen: Günde 1,000-5,000 mesaj arası (işletme büyüklüğüne göre)

## WhatsApp Toplu Mesaj İçin 72 Saatlik Free Entry Point Kuralı (Kritik!)

**Herkesin kaçırdığı nokta #1:** WhatsApp'ın en güçlü ama en az bilinen özelliği — müşteri size ilk mesaj attıktan sonra 72 saat boyunca:

- Tüm mesajlar ÜCRETSİZ (template olsun olmasın)
- İstediğiniz kadar mesaj atabilirsiniz
- Sadece 72 saat sonrası için template message ücreti ödenir

**Pratik uygulama — Abandoned Cart Recovery:**
```
0 dk:  Müşteri sepeti terk etti → otomatik WhatsApp mesajı (ücretsiz!)
1 saat: Hatırlatma mesajı (ücretsiz!)
24 saat: "Hâlâ sepetinizde ürün var" (ücretsiz!)
48 saat: "%10 indirim kodu" teklifi (ücretsiz!)
72 saat: Son hatırlatma — template message başlar (ücretli)
```

Bu strateji e-ticaret için en yüksek ROI'li WhatsApp otomasyonu. Ortalama %10-15 ek dönüşüm artışı.
