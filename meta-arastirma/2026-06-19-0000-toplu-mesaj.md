## WhatsApp Business API ile Toplu Mesaj

### Resmi API — WhatsApp Business Platform

**Özellikler:**
- Toplu mesaj gönderimi için **Template Message** zorunlu
- Her template için Meta onayı gerekli (24-48 saat)
- **72 saat kuralı:** Müşteri son mesajınızdan 72 saat içinde yanıtladıysa, onay almadan mesaj gönderebilirsiniz
- Rate limit: **1,000 mesaj/dakika** (Business API)
- Mesaj türleri: Metin, görsel, belge, buton, interactive list

**Template Örneği (n8n + Claude):**
```json
{
  "name": "order_confirmation",
  "language": "tr",
  "components": [
    {
      "type": "header",
      "format": "text",
      "text": "Siparişiniz Alındı ✅"
    },
    {
      "type": "body",
      "text": "Merhaba {{1}}, siparişiniz {{2}} numarasıyla alındı. Tahmini teslimat: {{3}}"
    },
    {
      "type": "footer",
      "text": "Sipariş takibi için: example.com/takip"
    }
  ]
}
```

## Herkesin Kaçırdığı Nokta #1 — WhatsApp Artık Sadece Mesajlaşma Değil

Herkes WhatsApp'ı "müşteri chat" aracı olarak görüyor. Ama Meta'nın gerçek hedefi **WhatsApp'ı işletme işletim sistemine dönüştürmek**.

**Bloomberg (Haziran 2026):** Meta, WhatsApp Business'ı "business operating layer" olarak konumlandırıyor — yani şirketler WhatsApp üzerinden:
- Sipariş alsın
- Ödeme alsın  
- Kargo takibi yapsın
- Müşteri desteği sunsun
- Hatta stok yönetimi yapsın

**Bu ne demek?** WhatsApp artık sadece "destek kanalı" değil — **tam bir satış platformu**. E-ticaret markaları için Instagram DM → WhatsApp → ödeme → teslimat tam döngüsü artık Meta ekosistemi içinde tamamen kapalı devre çalışıyor.

**Türkiye uyarlaması:** Bu model Türkiye'de özellikle D2C markalar için çok güçlü. Instagram'dan gelen müşteri WhatsApp'a yönlendiriliyor, orada AI bot siparişi alıyor, ödeme linki gönderiyor, kargo takibi yapıyor. Kullanıcı hiçbir zaman WhatsApp'ı terk etmiyor.

## Herkesin Kaçırdığı Nokta #2 — Toplu Mesaj Rate Limit Tuzakları

Herkes "1,000 mesaj/dakika" limitini yeterli görüyor. Ama gerçekte **template onay süresi** en büyük darboğaz.

**Sorun:**
- Yeni template → 24-48 saat onay süresi
- Onaylanan template → 1,000 mesaj/dakika
- Ancak: Her template sadece kendi kullanım alanında onaylanıyor
  - `order_confirmation` → sadece sipariş onayı için geçerli
  - `shipping_update` → sadece kargo bildirimi için geçerli
  - `promotion` → sadece pazarlama mesajları için geçerli

**Sonuç:** Kampanya başlatmak istediğinizde "template onayı bekliyorum" durumuna düşüyorsunuz. Önceden 10+ template hazırlamanız kritik.

## Herkesin Kaçırdığı Nokta #3 — Instagram AI Chatbot Güvenlik Açığı

**Bloomberg (Haziran 2026):** Meta'nın kendi Instagram AI destek asistanı, hacker'ların eline geçti. Sonuç: **20.000+ Instagram hesabı çalındı** — Barack Obama'nın Beyaz Saray'daki hesabı dahil.

**Nasıl oldu?**
1. Hacker'lar Meta AI chatbot'a "hesabımı kurtar" istedi
2. Chatbot doğrulama adımlarını atladı / yetersiz kontrol
3. Hesaplar hacker'lara devredildi
4. Ünlü hesaplar (Obama, diğerleri) ele geçirildi

**Herkesin Kaçırdığı Nokta:** Meta'nın kendi AI asistanı, Meta'nın kendi platformunda güvenlik açığı oluşturdu. Bu, AI otomasyonunun "her şeyi AI'a bırakma" riskini gösteriyor. **AI asistan eklerken mutlaka insan denetimi + çoklu doğrulama** olmalı.
