## WhatsApp Business API ile Toplu Mesaj

### Resmi API — WhatsApp Business Platform

**Özellikler:**
- Toplu mesaj gönderimi için **Template Message** zorunlu
- Her template için Meta onayı gerekli (24-48 saat)
- **72 saat kuralı:** Müşteri son mesajınızdan 72 saat içinde yanıtladıysa, onay almadan mesaj gönderebilirsiniz
- Rate limit: **1,000 mesaj/dakika** (Business API)
- Mesaj türleri: Metin, görsel, belge, buton, interactive list

**Template Örneği (n8n + Claude):**
```json
{
  "name": "order_confirmation",
  "language": "tr",
  "components": [
    {
      "type": "header",
      "format": "text",
      "text": "Siparişiniz Alındı ✅"
    },
    {
      "type": "body",
      "text": "Merhaba {{1}}, siparişiniz {{2}} numarasıyla alındı. Tahmini teslimat: {{3}}"
    },
    {
      "type": "footer",
      "text": "Sipariş takibi için: example.com/takip"
    }
  ]
}
```

## Kaçırılan Nokta #1 — 72 Saat Kuralının Gerçek Kullanımı

**Herkesin bildiği:** WhatsApp'ta 24 saat içinde yanıt vermeniz gerekiyor.

**Ama gerçekte ne var:** 72 saat kuralı aslında **session window** başlatıyor. Müşteri bir kez mesaj attığında, 72 saat boyunca o müşteriye istediğiniz kadar mesaj atabilirsiniz — her mesaj için ayrı template onayı gerekmez.

**Pratik kullanım:**
1. Müşteri soru soruyor → 72 saat başlıyor
2. Bu 72 saat içinde: sipariş onayı, kargo takibi, ürün önerisi, upsell — hepsi serbest
3. 72 saat dolmadan → "Sorunuz var mı?" gibi bir mesajla yeniden 72 saat açılır

**n8n ile otomatik tetikleme:** Müşterinin son mesaj saatini kaydet, 70. saatte otomatik "Size yardımcı olabilecek bir şey var mı?" mesajı gönder.

## Kaçırılan Nokta #2 — Toplu Mesaj = Kişiselleştirilmiş Mesaj

**Yanlış anlaşılma:** "Toplu mesaj = herkese aynı mesaj = düşük dönüşüm"

**Gerçek:** Modern toplu mesaj araçları CSV'deki alanları placeholder olarak kullanıyor:

```csv
 telefon,ad,urun,teslimat_gunu
 +905551234567,Ahmet, протеин tozu,3
 +905551234568,Ayşe,kreatin,5
```

Mesaj template'i:
```
Merhaba {{ad}}, {{urun}} siparişiniz {{teslimat_gunu}} iş günü içinde kargoda 🚚
```

Sonuç: Herkes kendi adına özelleştirilmiş mesaj alıyor, ama tek tek yazmak zorunda değilsiniz.

## Kaçırılan Nokta #3 — WhatsApp Web Otomasyonu = Meta TOS İhlali

**Önemli uyarı:** WhatsApp'ın kendi uygulamasını kullanarak toplu mesaj göndermek (WhatsApp Web otomasyonu, browser bot) **Meta Terms of Service'a aykırı**. Hesabınız kalıcı olarak banlanabilir.

**Güvenli alternatif:** WhatsApp Business API (resmi) veya yukarıdaki açık kaynak araçları kişisel kullanım içindir, ticari kullanımda risk sizin.

**Ticari kullanım için:** WhatsApp Business Platform onayı alın + Meta Business API kullanın.
