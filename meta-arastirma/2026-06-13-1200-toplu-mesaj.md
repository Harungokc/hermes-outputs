## 2. WhatsApp Business API — Resmi Yol

### Nedir?
WhatsApp Business Platform Cloud API, Meta'nın resmi toplu mesaj gönderme API'sidir. Kurumsal ölçekte mesajlaşma sağlar.

### Özellikler
- Programatik mesaj gönderimi (text, rich media, interactive messages)
- Sipariş onayları ve kargo güncellemeleri
- Randevu hatırlatmaları
- Çok dilli müşteri desteği
- Grup konuşmaları oluşturma ve yönetme

### Önemli Kısıtlamalar
- **Template Message Zorunluluğu:** Toplu mesajlar için önceden onaylatılmış template message gerekir. Bu template'ler "HSN" (Highly Structured Message) kategorisinde olmalı.
- **Rate Limit:** Her hesap için günlük ve saatlik mesaj limiti vardır. Aşım durumunda hesap askıya alınabilir.
- ** spam şikayeti durumunda:** Hesap kapatılabilir.

### Fiyatlandırma
- Mesaj başına ücretlendirme (hedef ülkeye göre değişir)
- Türkiye: ~$0.05-0.10/mesaj

### Kaynak
- [WhatsApp Business Platform Documentation](https://developers.facebook.com/docs/whatsapp/overview)
- [Cloud API Reference](https://developers.facebook.com/docs/whatsapp/cloud-api)

## 4. Herkesin Kaçırdığı Nokta #1

**WhatsApp Business API Template Message kuralı aslında bir FIRSAT.**

Herkes "toplu mesaj yasak" diyor ama gerçek şu: Eğer mesajınız bir template'e uygunsa (sipariş güncelleme, randevu hatırlatma, shiping update), Meta bu mesajları TAM OLARAK DESTEKLİYOR. Yani e-ticaret sitenizdeki "siparişiniz kargoya verildi" bildirimi, WhatsApp Business API ile tamamen yasal ve istenmeyen spam kategorisine GİRMİYOR.

**Sırr:** Abandoned cart hatırlatması = "Utility message" kategorisinde → onay alınca gönderilebilir, ban riski YOK.

---

## 5. Herkesin Kaçırdığı Nokta #2

**"Bulk messaging" terimi yanıltıcı.**

WhatsApp "bulk" kelimesini yasaklamış gibi görünse de aslında kural şu:aynı mesajı farklı kişilere göndermek DEĞİL, kişiselleştirilmiş otomatik yanıtlar GÜVENLİ. n8n + WhatsApp Business API ile her müşteriye özel mesaj atılabilir — bu "bulk" değil, "automated personalized messaging" olarak geçer.

**Örnek:** Sepeti terk eden müşteriye "Merhaba [isim], sepetinizde [ürün] bekliyor. Hemen tamamlayın: [link]" — kişiselleştirilmiş, ban riski sıfır.

## 6. Adım Adım Yapım Rehberi

### Resmi Yol: WhatsApp Business API + n8n

** Gerekenler:**
1. WhatsApp Business Platform hesabı
2. Meta Business Manager hesabı
3. n8n kurulumu
4. WhatsApp Business API erişimi (onay süreci ~1-2 hafta)

**Adımlar:**

1. **Meta Business Manager'da uygulama oluştur:**
   - developers.facebook.com → yeni uygulama
   - Ürün ekle: WhatsApp Business

2. **WhatsApp Business API Setup:**
   - Telefon numarası ekle (yeni veya mevcut)
   - Template message oluştur (onay ~24 saat)
   - API key al

3. **n8n Workflow:**
   ```
   [Trigger: Sipariş/Abandoned Cart]
   → HTTP Request (WhatsApp API)
   → Template ID gönder
   → Müşteri telefon numarası
   → Gönder
   ```

4. **İdeal Kullanım Senaryoları:**
   - Sipariş onayı → otomatik
   - Kargo güncelleme → otomatik
   - Randevu hatırlatma → otomatik
   - Terkedilen sepet → 1s, 24s, 72s delay ile
