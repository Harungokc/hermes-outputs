## Detaylı Analiz

### Test 1: Browser Bike Game
- Her iki model de Codex/Claude Code üzerinden oyun geliştirdi
- Fable daha yaratıcı tasarım kararları aldı
- Sol daha hızlı tamamladı

### Test 2: Scroll-Stopping Website
- Landing page + animasyonlar
- Fable daha iyi UX kararları
- Sol kodu hızlı shipping etti

### Test 3: Five Visual Worlds
- Karmaşık görsel projeler
- Fable = daha rafine çıktılar
- Sol = yüksek hacim, ortalama kalite

### API One-Off Tests
- Hız: Sol 2-3x daha hızlı
- Maliyet: Sol ~60% daha ucuz
- Güvenilirlik: Fable daha tutarlı

### Final Verdict
- Fable = Manager model (stratejik, yaratıcı)
- Sol = Worker model (hızlı, ucuz, agentic)
- Hybrid yaklaşım en mantıklı
