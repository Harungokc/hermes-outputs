## Detaylı Analiz

### 1. Modeller vs Context (Çekirdek Analoji)
- Chat = sohbet, Code = kod yazma, Cowork = iş ortağı
- Context window = AI'ın kısa süreli hafızası
- Memory = kalıcı bilgi deposu
- Model seçimi任务'a göre değişir

### 2. 12 Mindset Shift
- "Yapmasını istemek" → "Sonucu tanımlamak"
- "Kod yazdırmak" → "AI'ı yönlendirmek"
- "Tek başına çalışmak" → "AI ekibi kurmak"

### 3. AI Operating System Kurulumu
- claude.md = proje kuralları
- settings.json = global tercihler
- .env = API anahtarları
- Google Workspace MCP entegrasyonu

### 4. Skills (Beceriler) Deep Dive
- Beceriler = tekrar kullanılabilir prompt'lar
- SKILL.md formatı
- Örnek: "Grill-Me" becerisi

### 5. Sub-Agents (Alt-Ajanlar)
- Alt-ajan = bağımsız çalışan AI
- Plan Roaster örneği
- Güven seviyeleri ve permission modları

### 6. Second Brain (İkinci Beyin)
- 5 seviye: Dosyalama → Özetleme → Analiz → Senaryo → Karar
- Knowledge graph yapısı
- Session handoff = hafıza aktarımı

### 7. Token Management
- Prompt caching = maliyet düşürür
- Model seçimi = hız/maliyet dengesi
- Working context = ne kadar bilgi verilmeli

### 8. Scheduled Automations
- Cloud routines = cron job benzeri
- Research automation örneği
- Güven + doğrulama döngüsü
