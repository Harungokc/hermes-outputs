## Nate Herk Odak Noktası
Nate Herk bu hafta **agent loop** kavramını sadeleştiriyor. GLM 5.2 ile open-source model kullanımı, Claude Code içinde alternatif model routing konusunda öncü. "Agent Loops" videosu bugün yayınlanmış ve agent engineering'i herkes için erişilebilir hale getiriyor.

**Herkesin Kaçırdığı Nokta:** Agent loop'ların başarısı architecture değil, **verification step** — yani agent'ın kendi çıktısını kontrol edebilmesi. Çoğu kişi 5 agent çalıştırmaya odaklanıyor, ama asıl mesele "done criteria" belirlemekte.
