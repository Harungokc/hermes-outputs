## 1. Intuned (YC S22) — Browser Otomasyonu Artık "Kod" Olarak Çalışıyor

**Herkes Ne Konuşuyor:** YC Summer 2026'dan çıkan yeni browser otomasyon aracı. "Build and run reliable browser automations as code" sloganıyla HN'de 117 puan almış.

**Ama Asıl İlginç Olan:** Çoğu browser otomasyon aracı ya selector-based ya da AI-vision-based çalışır. Intuned ikisini birleştiriyor: Kod olarak tanımlanmış workflow'lar, arka planda LLM destekli element detection ile çalışıyor. Yani "butona tıkla" yazmak yetiyor, XPath yazmak zorunda değilsin. Ayrıca "reliable" vurgusu rastgele değil — mevcut araçlar %30-40 oranında flaky (güvenilmez) sonuçlar veriyor. Intuned bunu düzeltmek için built-in retry + state verification katmanı sunuyor.

**Teknik detay:** Playwright/Puppeteer tabanlı, ama API'si tamamen imperative kod yerine declarative config istiyor. "Bunu yap, sonra şunu bekle, sonra kontrol et" döngüsü built-in.

## 4. KPMG + Microsoft Agent 365 — Enterprise AI Governance Artık Global Ölçekte

**Herkes Ne Konuşuyor:** KPMG ve Microsoft, Agent 365'ı global ölçekte dağıtmaya başladı. Yani kurumsal AI agent'lar artık sadece "pilot proje" değil, tam ölçekli üretimde.

**Ama Asıl İlginç Olan:** Bugüne kadar AI agent deployment'ları "ya hep ya hiç" durumdaydı — ya tam otomasyon (riskli) ya tam insan kontrolü (pahalı). KPMG + Microsoft'un yaptığı şey "governance layer" eklemek: Agent ne yapabilir, kime sorabilir, hangi veriye erişebilir — hepsi Microsoft Purview entegreli. Asıl mesele: Agent'lar artık "yapması gerekeni yapıyor" değil, "izin verilen sınırlar içinde yapması gerekeni yapıyor" moduna geçiyor. Bu ayrım enterprise için kritik — çünkü regulator'lar (SOC 2, GDPR) agent decision'larını trace edilebilir istiyor.

**Metrik:** Agent 365 governance katmanı, her agent kararını %100 logluyor — bu enterprise müşteriler için en büyük güven unsuru.
