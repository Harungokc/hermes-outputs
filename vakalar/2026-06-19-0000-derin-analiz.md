**Nasıl yapılır:**

3. Policy ve sandbox kuralları koy
4. Real-time dashboard'dan takip et
**İhtiyaçlar:** Python 3.10+, Multiple AI agent hesapları, OpenAI/Anthropic API key

### 2. Ponytail — Agent Over-Engineering Önleme
**Neden popüler:** AI agent'lar gereksiz kod yazma eğiliminde. Ponytail bunu engelliyor — sadece gerekli olanı yaz, geri kalanı sil.
**Kime lazım:** Her AI coding agent kullanıcısı — özellikle junior developer'lar
**Nasıl yapılır:**
1. Claude Code kullanıyorsan `ponytail` plugin'ini kur
2. Agent'a "tembel senior dev" mindset'ini aktar
3. Her kod bloğu için "bunu yazmasam olur muydu?" sorusunu sor
**İhtiyaçlar:** Claude Code, Node.js (plugin kurulumu için)

### 3. Harness Anything — Ofis Otomasyonu
**Neden popüler:** Excel/Word otomasyonu = en yüksek ROI. Her şirket Excel kullanıyor. AI agent ile Excel automation = huge time savings.
**Kime lazım:** Finans, muhasebe, HR, veri girişi yapan ofis çalışanları
**Nasıl yapılır:**
1. Windows'ta WPS veya MS Office kur
2. `pip install harness-anything` ile kur
3. CLI komutlarıyla Word/Excel dosyalarını AI agent'a kontrol ettir
4. n8n ile workflow'a bağla
**İhtiyaçlar:** Windows OS, WPS/MS Office, Python 3.10+

### 4. Guard Skills — AI Kod Kalitesi
**Neden popüler:** AI üretimi kod hatalı olabiliyor. İnsan code review zaman alıyor. Guard Skills ile otomatik quality gate.
