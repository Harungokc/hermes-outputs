## ✅ 5 TAVSİYE

### 1. taste-skill
**Neden popüler:** "AI generic konuşuyor" şikayeti evrensel. Herkesin kaçırdığı nokta: Skill sadece "daha iyi yaz" demiyor — agent'ın persona'ını, karar mekanizmasını ve output formatını değiştiriyor.

**Kime lazım:** İçerik üreticileri, satış pazarlama profesyonelleri, LinkedIn post yazarları

**Nasıl yapılır:**
1. `git clone` repoyu al
2. Claude Code'a skill'i yükle (readme'deki komutları takip et)
3. Agent'a "Use taste-skill for all outputs" prompt'u ver

**İhtiyaçlar:** Claude Code, taste-skill repo

---

### 2. last30days-skill
**Neden popüler:** Pazar araştırması = her işin başlangıcı. Kimse 6 platformu tek tek gezmez. Bu skill tek komutla hepsini tarıyor.

**Kime lazım:** Yatırımcılar, girişimciler, ürün yöneticileri, rekabet analizi yapanlar

**Nasıl yapılır:**
1. `git clone` repoyu al
2. API key'leri (.env) ayarla
3. `claude-code --skill last30days` ile araştırma başlat

**İhtiyaçlar:** Claude Code, Reddit/X/YouTube API key'leri (veya ücretsiz mod)

---

### 3. oh-my-claudecode
**Neden popüler:** "6 kişilik ekip" etkisi — tek Claude Code yerine çoklu agent sistemi. Her agent farklı uzmanlık alanında çalışıyor.

**Kime lazım:** Solo founder'lar, küçük startup'lar, 1-5 kişilik ekipler

**Nasıl yapılır:**
1. `git clone` repoyu al
2. `npm install && npm run setup`
3. Multi-agent mode'u etkinleştir
4. Projeyi agent ekibine ver

**İhtiyaçlar:** Node.js, Claude Code, oh-my-claudecode repo

---

### 4. marketingskills
**Neden popüler:** AI coding agent'lar kod yazabiliyor ama "conversion odaklı içerik" üretemiyor. Bu skill onları gerçek bir growth engineer'e dönüştürüyor.

**Kime lazım:** SaaS şirketleri, e-ticaret markaları, dijital pazarlamacılar

**Nasıl yapılır:**
1. `git clone` repoyu al
2. Claude Code'a marketingskills'i ekle
3. "Run CRO audit on [landing page]" gibi komutlar ver

**İhtiyaçlar:** Claude Code, marketingskills repo, analiz edilecek web sitesi

---

### 5. karpathy/autoresearch
**Neden popüler:** Andrej Karpathy'nin ismi + tek GPU'da çalışan otomatik araştırma. Herkesin kaçırdığı nokta: "Büyük AI araştırması = büyük maliyet" efsanesini çürütüyor.

**Kime lazım:** AI araştırmacıları, ML mühendisleri, üniversite öğrencileri

**Nasıl yapılır:**
1. `git clone https://github.com/karpathy/autoresearch`
2. Tek GPU'lu makineye kur (最低 RTX 3090 / A100)
3. `python train.py --topic "AI agent trendleri"`
4. Sonuçları markdown olarak al

**İhtiyaçlar:** Python, PyTorch, tek GPU (min 24GB VRAM), CUDA
