## 1. Xiaomi MiMo-Code — 10,336 ⭐

### Ne Yapıyor?
Xiaomi'nin terminal-native AI coding assistant'ı. Claude Code, Codex gibi diğer agent'lardan farkı: **"Models and Agents Co-Evolve"** yaklaşımı. Agent sadece kod yazmıyor — model ile birlikte evriliyor, birbirini sürekli iyileştiriyor.

### Teknik Detaylar
- **Kurulum:** `curl -fsSL https://mimo.xiaomi.com/install | bash` — tek satır
- **Alternatif:** `npm install -g @mimo-ai/cli`
- **Kanallar:** MiMo Auto (ücretsiz, sınırlı süre), Xiaomi MiMo Platform (OAuth), Claude Code migration, custom OpenAI-compatible API
- **Dil:** TypeScript
- **Topics:** ai, ai-agents, cli, mimo, mimo-code
- **Oluşturulma:** 2026-06-10

### Herkesin Kaçırdığı Nokta
**"Co-evolution" = tek yönlü agent'dan farklı.** Mevcut AI coding agent'lar (Claude Code, Codex, Cursor) modelin kapasitesini kullanıyor — agent modelin output'unu tüketiyor. MiMo-Code'ta ise agent'ın çalışması modelin kendisini değiştiriyor. Agent bir hata yapıyor → model bunu görüyor → bir sonraki generation'da düzeltiyor. Bu, 2026'nın en iddialı "model improvement via agent work" yaklaşımı.

**İkinci kaçırılan:** Xiaomi = sadece telefon şirketi değil. 10,000+ çalışan Ar-Ge ekibi, HyperOS ecosystem'ü, IoT cihazları. MiMo-Code bu ecosystem'ün içinden çıkıyor — agent'ın "gerçek dünya" ile bağlantısı (IoT, mobil, giyilebilir) diğer agent'larda yok.

**Üçüncü kaçırılan:** Ücretsiz kanal = Çin'in Batı AI agent'larına cevabı. Claude Code Batı'da yaygınken, MiMo-Code Çinli geliştiriciler için ücretsiz erişim sağlıyor. Bu, gelecekte "Çin vs Batı AI coding agent" paradigm split yaratabilir.
