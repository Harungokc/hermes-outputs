## 🔟 EN İLGİNÇ 10 PROJE

### 1. OpenAI Codex — Beyaz Yakalı İş İçin Yeni Agent Yetenekleri
- **Kim:** OpenAI
- **Ne:** OpenAI, Codex için yeni yetenekler duyurdu — beyaz yakalı ofis işleri için tasarlanmış agent araçları. Doc generation, spreadsheet analizi, email otomasyonu.
- **Tech:** Codex (GPT-4 tabanlı), code interpreter, function calling
- **Sayı:** Kurumsal kullanıcı sayısı açıklanmadı ama OpenAI'nin en büyük update'i
- **Neden çıkıyor:** OpenAI artık sadece "coding agent" değil — "office worker agent" yapıyor. Word, Excel, PowerPoint otomasyonu demek. Microsoft 365 ile doğrudan rekabet.

### 2. Nvidia + Microsoft RTX Spark — Yerel AI Agent PC'leri
- **Kim:** Nvidia × Microsoft ortaklığı
- **Ne:** RTX Spark PC'ler — yerel (local) AI agent'lar için optimize edilmiş Windows bilgisayarları. Cloud'a bağlı kalmadan agent çalıştırma.
- **Tech:** Nvidia RTX GPU, Windows 11, local LLM inference
- **Sayı:** Büyük PC üreticilerinden (Dell, HP, Lenovo) çıkacak sistemler
- **Neden çıkıyor:** **Herkesin kaçırdığı nokta:** "Cloud AI agent" güvenlik endişesi olan şirketler için local agent = veri şirket dışına çıkmaz. Büyük kurumsal IT satışı için ideal. Ama maliyet $1,000+ — tüketici pazarı için değil.

### 3. Microsoft AI Wearable Badge — ofis Çalışanları İçin İzleme
- **Kim:** Microsoft
- **Ne:** Ofis çalışanları için AI destekli wearable badge. Toplantı özeti çıkarıyor, konuşmaları analiz ediyor, verimlilik ölçüyor.
- **Tech:** AI-powered audio processing, Microsoft 365 entegrasyonu
- **Sayı:** Genel kullanıma açık
- **Neden çıkıyor:** **Herkesin kaçırdığı nokta:** Microsoft "toplantı agent'ı" yapıyor — ama kimse bunu wearable badge olarak beklemedi. Çalışan izleme endişesi yaratacak. Privacy aktivistlerinden büyük tepki beklenir.

### 4. Mistral Vibe — Endüstriyel AI Agent'a Giriş
- **Kim:** Mistral AI (Fransa)
- **Ne:** Mistral, "Vibe" adlı yeni model ailesini duurdu — endüstriyel AI uygulamalarına odaklı. Aynı zamanda data center yatırımı yapıyor.
- **Tech:** Mistral models, industrial AI, data center infrastructure
- **Sayı:** Avrupa'nın en büyük AI startup'ı
- **Neden çıkıyor:** OpenAI'ye karşı Avrupa alternatifi. Fransa'nın "AI sovereignty" hamlesi. Avrupa şirketleri için veri lokalizasyonu avantajı.

### 5. DoorDash AI Chatbot — Fotoğraftan Sipariş
- **Kim:** DoorDash
- **Ne:** Kullanıcılar fotoğraf çekip "bunu sipariş et" diyebiliyor. AI yemeği tanıyor, restoranda arıyor, sipariş veriyor.
- **Tech:** Computer vision, conversational AI, restaurant integration
- **Sayı:** DoorDash'in en büyük AI lansmanı
- **Neden çıkıyor:** **Herkesin kaçırdığı nokta:** "Vision + agent" kombinasyonu ilk kez yaygın bir uygulamada. Daha önce sadece demo'larda gördük — şimdi gerçek üründe. E-ticaret için product recognition + order agent = yeni kategori.

### 6. Anthropic Claude Opus 4.8 — Coding + Honesty İyileştirmeleri
- **Kim:** Anthropic
- **Ne:** Claude Opus 4.8 — coding agent yeteneklerinde ve "honesty" (dürüstlük) metriklerinde iyileştirme. Daha az hallucination, daha doğru kod.

- **Neden çıkıyor:** **Herkesin kaçırdığı nokta:** Anthropic "hallucination" ile savaşmayı coding performansından önce koyuyor. Diğer şirketler hız derken, Anthropic güvenilirlik üzerine kuruyor. Enterprise müşterileri için doğruluk = her şey.

### 7. Google Antigravity 2.0 — $100 AI Ultra Plan

### 8. NewCore $66M — AI Agent Identity Security

### 9. Hyper (YC P26) — Company Brain for Agentic Development — 79⭐

- **Tech:** YC startup, company knowledge base, agentic development
- **Sayı:** 79 HN puan, YC P26 batch
- **Neden çıkıyor:** **Herkesin kaçırdığı nokta:** "Her şirketin kendi AI agent'ı olacak" demesi kolay — ama agent'lar neyi bilecek? Hyper = agent'ların şirketi "öğrenmesi" için altyapı. Bu problem çözülmeden agent'lar yarım güçle çalışıyor.
